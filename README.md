# E-commerce Sales Data Analysis

A data analytics project focused on exploring retail sales data to identify customer behaviour patterns, category performance and business trends through statistical analysis and visualisation.

## Technologies Used

- Python
- pandas
- NumPy
- Matplotlib
- Excel

## Project Objectives

- Analyse sales performance across categories
- Identify customer purchasing trends
- Explore revenue and profitability patterns
- Generate business-focused recommendations

## Analysis Performed

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualisation
- Correlation analysis
- Statistical testing
- Business insight generation

## Key Outcomes

- Identified category-level sales trends and customer purchasing patterns
- Produced visual dashboards and performance summaries
- Generated recommendations to support business decision-making
- Demonstrated practical application of data analytics techniques

## Academic Context

Developed as part of a university Data Science module, achieving a mark of 87%.
